# 🌌 Aura Discover: The Antigravity AI Directory

Aura Discover is a premium, high-performance AI tools directory built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**. It features a futuristic "Antigravity" UI where tool cards float dynamically, providing a unique and interactive user experience.

## ✨ Key Features

- **🚀 Antigravity UI:** Physics-based floating animations for tool cards using Framer Motion.
- **🧠 "Find My AI" Wizard:** An interactive, multi-step recommendation engine that helps users choose the right tool based on their goals, budget, and expertise.
- **🔍 Real-time Search & Filtering:** Instant filtering by categories and keywords with synchronized URL state management.
- **🌙 Premium Dark Theme:** A sleek "Linear/Vercel" inspired aesthetic with glassmorphism, glowing borders, and high-contrast typography.
- **📱 Fully Responsive:** Optimized for everything from mobile screens to ultra-wide monitors.
- **❤️ Favorites System:** Save your favorite AI tools locally (powered by `localStorage`).
